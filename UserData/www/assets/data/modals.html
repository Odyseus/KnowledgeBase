About

  • Website created with Bootstrap, HTML, CSS, and JS library.

  • Bootstrap theme created with Bootstrap themes generator and based on Bootswatch's Flatly theme.

  • Code syntax highlighting powered by highlight.js.

  • The index uses DataTables table enhancing plug-in to display, sort and search content.

  • Markdown files are rendered inline using mistune, a Python Markdown parser with renderers and plugins.

  • Glyph icons are provided by Nerd Fonts, iconic font aggregator, collection, and patcher.
